Javascript Conditionals

Conditional statements control behavior in JavaScript and determine whether or not pieces of code can run.

There are multiple different types of conditionals in JavaScript including:

  1. “If” statements: where if a condition is true it is used to specify execution for a block of code.
  2. “Else” statements: where if the same condition is false it specifies the execution for a block of code.
  3. “Else if” statements: this specifies a new test if the first condition is false.

Switch Case Condition

Switch is a type of conditional statement that will evaluate an expression against multiple possible cases and execute one or more blocks of code based on matching cases. The switch statement is closely related to a conditional statement containing many else if blocks, and they can often be used interchangeably.

Switch Case Example

var text;
var fruits = document.getElementById("myInput").value;

switch(fruits) {
case "Banana":
text = "Banana is good!";
break;
case "Orange":
text = "I am not a fan of orange.";
break;
case "Apple":
text = "How you like them apples?";
break;
default:
text = "I have never heard of that fruit...";
}
